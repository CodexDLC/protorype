# tools/static/

Сборка статических ресурсов.

## css_compiler.py

Компилирует `base.css` в `app.css`: раскрывает все `@import`, убирает комментарии.

```bash
python tools/static/css_compiler.py
```

**Что делает:**
1. Читает `src/backend_django/static/css/base.css`
2. Рекурсивно раскрывает все `@import url(...)` — встраивает содержимое
3. Удаляет комментарии (`/* ... */`)
4. Пишет результат в `src/backend_django/static/css/app.css`

**Режимы:**
- По умолчанию: удаление комментариев, форматирование сохраняется
- `minify=True`: полная минификация (пробелы, переносы)

Генерированный `app.css` — единственный файл который нужно подключать в шаблонах.
`base.css` — только для разработки, в git не игнорируется.
